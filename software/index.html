Software

finding good programs is a challenging endeavor. sometimes you just want to do something quickly, and you end up finding a website or a piece of software which is so bad that it is unusable.

this page lists out software and programs which i use, and that are open to reuse, modify and copy for your own needs.

basic

=====

operating system – use any distro 1 with the penguin inside.

if you are a beginner, ubuntu is a popular option. i started with that and now i use arch. i do not have a strong opinion on distros, however personally, i like a debian-based or an arch-based system.

worth a shot » manjaro with xfce D.E.


window managers – my choice is dwm. a dynamic window manager for X. it is written in C, and it is very fast. for a status monitor i use slstatus.

although it is not for everyone, by far, this has been my favorite thing to use.

web browsers – firefox and brave with tweaks. although both are free and open-source, they have some proprietary blobs, collect telemetry, and have some settings enabled by default.

i use and recommend these add-ons to make the browsing experience better:

– uBlockOrigin - essential for blocking ads, trackers and other stuff.
– privacy badger - something i always install after uBlock.
– containers - compartmentalizes online activity. brave profiles useful too.

terminal & shell – alacritty and st.2. if you are in a server, you typically will find xterm as a default; make sure you install tmux. it is a must have.

zsh as my shell. it is bash, but with additional features.

text editor/IDE – vim/neovim to edit something very fast. vscodium is good too. pycharm has a lot of features, but it comes with its own limitations like memory consumption. jupyter notebook for data analysis.

utilities

file manager – thunar / pcmanfm (GUI) and ranger (CLI)
media player – mpv. there’s no going back once you start using it.
app launcher – dmenu
password management * keepassXC - offline * bitwarden - cloud.
mail client – thunderbird.
pdf viewer - mupdf - best pdf viewer i found.
rss feeds – newsboat. cli-based. if you want updates from your fav sites.
